DESCRIPTION

t.pytorch.predict is a wrapper around the i.pytorch.predict module and supports all relevant flags and options of that module. t.pytorch.predict compiles the input imagery groups to i.pytorch.predict from the temporal granules in the input STRDS. Those groups pre granule are complemented with raster maps from a static_group and/or another reference_strds, where maps in the reference STRDS are matched with the input STRDS in space and time using the user-defined sampling. If a reference STRDS or a static group is used it often makes sense to provide a basename for the resulting raster maps.

For more information on how machine learning models are applied to the imagery groups, please consult the manual of i.pytorch.predict.

EXAMPLES

Predict FSC for a Sentinel-3 time series

  t.pytorch.predict input=/data/Sentinel3/SLSTR/ output=Sentinel_3_SLSTR_RBT_L2 \
  nprocs=8 (...)

t.info Sentinel_3_SLSTR_RBT_L2

DESCRIPTION

i.sentinel1.import pre-processes Sentinel-1 imagery accortding to a standard protocol outlined by <> and imports the resulting GeoTiff(s) into the current mapset.

A textfile for use with t.register can be requested. Per default, results are imported into the GRASS GIS database and temporary data is deleted after the process finishes, unless the user chooses to link the resulting GeoTiffs.

EXAMPLES

Import 

  i.sentinel1.import --v --o  directory=./ \
  dem=DTM_10m \  # output_directory= /sentinel1
  specle_filter=revised_lee \
  nprocs=2

REQUIREMENTS

The ESA SNAP software needs to be installed and it`s batch processing tool gpt needs to be executable and available on PATH.
